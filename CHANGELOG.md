# Resolución de los ejercicios

# Ejercicio 0

- Se ha aprendido a hacer un fork de un repositorio.
- Se ha aprendido a realizar varias ramas y realizar un merge de todas ellas en la rama `libro`.
- Se ha aprendido a resolver conflictos en un merge.

## Ejercicio 1 y 2

- Se ha aprendido como hacer un fork de un repositorio y a clonarlo en local.
- Se ha creado una rama nueva `resulotados` y se han hecho cambios en ella, además de eliminar la carpeta soluciones para hacerla nosotros sin ver nada.

## Ejercicio 3 y 4

- Se ha aprendido a aprendido a clonar la rama del ejercicio del github y a hacer un push de los cambios en la rama `libro`.
- Se ha aprendido a revisar que el repositorio este conectado correctamente y a hacer un pull de los cambios realizados en el repositorio remoto.

## Ejercicio 5

- Se ha aprendido a hacer una pull request de una rama a otra.
- Se ha tomados conciencia de selecionar las ramas correctas para hacer el merge para no afectar a las ramas de los compañeros.
- Se ha aprendido a realizar un merge de la rama `libro` a la rama `resultados`.
- Se ha cambiado el cambiado la rama por defecto a `resultados` para que se vea en el github.

## Ejercicio 6

- Se ha aprendido a hacer una Issue en github. **Importante: Solo deja hacerla en el repositorio original**. Por el momento no podemos porque no somos colaboradores del repositorio original.
- Se ha aprendido a enviar una PR a un proyecto ajeno.
- Se ha aprendido a hacer una Issue en un proyecto ajeno y a enviar una PR a un proyecto ajeno.
